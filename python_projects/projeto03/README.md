Jogo de Construção de Navios de Guerra
Desenvolvedor: Marcos Almeida Pereira Neto

**O que é?**
Este é um jogo de gerenciamento de recursos e tempo, onde você pode construir navios de guerra de diferentes classes. O jogo roda em uma janela gráfica usando a biblioteca Pygame.

**Como Jogar**
O tempo avança automaticamente. Pressione a barra de espaço para pausar e despausar o tempo.

A velocidade do tempo pode ser ajustada usando as seguintes teclas:
1: Velocidade normal (1 dia a cada 2 segundos)
2: Velocidade 2x (1 dia a cada 1 segundo)
3: Velocidade 4x (1 dia a cada 0.5 segundos)
4: Velocidade 20x (1 dia a cada 0.1 segundos)

**Requisitos**
Você precisará do Python 3.9 (ou superior) e da biblioteca Pygame.

Para instalar o Pygame, execute o seguinte comando no terminal:
pip install pygame

**Como Executar**
Clone este repositório ou baixe o arquivo main.py.
Abra um terminal na pasta do projeto.
Execute o jogo com o comando:
python main.py
